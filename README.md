# ShoppingListApp

Приложение для управления списком покупок, разработанное на .NET MAUI.

## Функциональность

- Добавление покупок (название, количество, категория)
- Отметка покупки как выполненной
- Удаление покупок
- Сохранение данных в JSON файл
- Загрузка данных при запуске

## Используемые технологии

- C#
- .NET MAUI
- System.Text.Json

## Тестирование

- Позитивный тест: добавление покупки работает корректно
- Негативный тест: пустое название не добавляется
- Граничный тест: одна покупка сохраняется и загружается

## Запуск

Открыть проект в Visual Studio  
Выбрать эмулятор или Windows  
Нажать F5
